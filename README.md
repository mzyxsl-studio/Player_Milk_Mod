## 开发环境
+ IntelliJ IDEA
+ Gradle 8.14.2
+ Java 21

## 构建

Windows

```cmd
git clone https://github.com/mzyxsl/Player_Milk.git
cd Player_Milk_Mod
gradlew genSources
gradlew build
```

Linux

```bash
git clone https://github.com/mzyxsl/Player_Milk.git
cd Player_Milk
chmod +x gradlew  # 首次执行需要给执行权限
./gradlew genSources
./gradlew build
```
